leanote安装-原生安装
2018-09-27 20:27:00 #linux #leanote

1、下载

下载网址：

http://leanote.org/#download 下载最新版本的leanote linux版本

2、安装MongoDB

leanote使用的是MongoDB作为数据库。

2.1 下载地址：

https://www.mongodb.com/download-center–>Communtity Server–>Linux–>RHEL 7 Linux 64-bit x64

2.2 配置环境变量

把MongoDB配置到环境变量，便于启动MongoDB，如果使用的不多，也可以不配置。

vim /etc/profile
export PATH=$PATH:/home/user1/mongodb-linux-x86_64-2.6.4/bin
source /etc/profile

2.3 启动MongoDB

2.4 简单使用MongoDB

3、leanote使用

3.1 初始化MongoDB的leanote数据

mongorestore -h localhost -d leanote –dir /home/user1/leanote/mongodb_backup/leanote_install_data/
执行之后MongoDB中便有leanote这个数据库

3.2 配置leanote

修改leanote 解压路径下的conf/app.conf文件中的app.secret的值，随便增删，网上说不改会有安全问题

3.3 运行leanote

3.4 界面

leanote默认的端口号是9000
在浏览器访问http://ip:9000即可看到界面
