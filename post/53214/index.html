2、安装

这里的安装教程主要是参考https://disconf.readthedocs.io/zh_CN/latest/install/src/02.html这个网站的，这个网站应该算是disconf的官方文档的，但是好像更新不是很经常，有不少错误。然后之所以要做一下笔记，主要是有一些东西和官方给的不一样，而且官方觉得我是天才，其实我啥也不懂，所以很多东西需要自己去搞定的。

事先准备

在安装disconf之前需要实现准备安装一些东西，如下：

  • zookeeper
  • redis
  • nginx
  • mysql
  • tomcat
  • maven
  • git

鉴于这些东西安装比较简单，可以自行安装。
